## Interactive Features {#features}

### AI Chatbot {#chatbot}

This book includes an embedded AI chatbot that can:

- **Answer general questions** about any topic in the book
- **Explain selected text** - highlight any text and ask for clarification
- **Provide citations** - all answers reference specific chapters and sections

Look for the chat icon in the bottom right corner of the screen.

### Personalization {#personalization}

At the start of each chapter, you'll find a **"Personalize this chapter"** button that:

- Adjusts explanations based on your experience level
- Provides appropriate examples (beginner vs. advanced)
- Modifies the pacing and depth of content

### Urdu Translation

Need content in Urdu? Each chapter includes a **"Translate to Urdu"** button that:

- Translates all text content to Urdu
- Preserves code blocks and technical terms
- Allows toggling back to English
