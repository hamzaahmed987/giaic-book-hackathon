# Chapter 6 Summary

## Architecture Overview

```
Frontend (Docusaurus) ←→ Backend (FastAPI) ←→ AI Services
                              ↓
                    Neon Postgres + Qdrant
```

## Key Components Built

| Component | Technology | Purpose |
|-----------|------------|---------|
| Frontend | Docusaurus + React | Book UI, Chatbot |
| Backend | FastAPI | API, Business Logic |
| Auth | Better-Auth | User authentication |
| Database | Neon Postgres | Users, sessions |
| Vector DB | Qdrant | Embeddings, RAG |
| AI | OpenAI API | LLM, Embeddings |

## Features Implemented

- RAG Chatbot with citations
- User authentication
- Profile-based personalization
- Urdu translation
- Content caching
